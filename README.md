# 💰 Sistema de Banco em Java

Projeto simples de **simulação de um sistema bancário no console**, desenvolvido em **Java** com foco em praticar conceitos de **Programação Orientada a Objetos (POO)**.

A ideia do projeto surgiu como exercício prático durante os estudos de Java, implementando funcionalidades básicas de uma conta bancária.

---

# 🚀 Funcionalidades

O sistema permite realizar operações básicas de uma conta bancária:

* 💵 **Depósito**
* 💸 **Saque**
* 📊 **Consulta de saldo**
* 📋 **Menu interativo no terminal**

Todas as interações acontecem diretamente pelo **terminal/console**.

---

# 🧠 Conceitos aplicados

Durante o desenvolvimento foram praticados conceitos importantes de Java:

* Programação Orientada a Objetos (POO)
* Separação de responsabilidades
* Uso de **Services**
* Uso de **Collections (Set)**
* Estruturas de controle (`if`)
* Interação com usuário usando **Scanner**

---

# 🏗 Estrutura do Projeto

O projeto está organizado em pacotes para manter o código mais organizado.

```
src
 ├─ application
 │   └─ Program.java
 │
 ├─ services
 │   └─ UserAccountService.java
```

* **Program** → responsável pela interação com o usuário.
* **Service** → contém a lógica das operações da conta.

# ▶️ Como executar o projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/gumohanna/sistema-banco.git
```

### 2️⃣ Abrir no IntelliJ / Eclipse / VSCode

Importe o projeto como um projeto Java.

### 3️⃣ Executar a classe principal

Execute a classe:

```
Program.java
```

O sistema irá iniciar no terminal.
